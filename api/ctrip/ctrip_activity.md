# 产品信息

### 描述

根据产品编号，批量获取途风产品信息


### API地址

    /ctrip/product/detail

### 参数

| 参数           | 类型          | 是否必须 | 描述             |
| -------------- |:-------------:| ----:| :-----------------|
| product_codes     | string  |  是   | 产品编号  |

### 参数示例

```javascript
{
	"product_codes":["4512-101580837"]
}
```
